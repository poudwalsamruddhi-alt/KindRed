KindRed

KindRed is a modern NGO-inspired social impact and awareness platform designed to encourage community participation and spread awareness about important social causes through a clean, responsive, and engaging web experience.

Live Demo

https://kind-red.vercel.app/



Features

- Modern NGO-inspired user interface
- Fully responsive design
- Volunteer participation section
- Donation awareness pages
- Programs and impact showcase
- Stories and annual report sections
- Climate, education, healthcare, nutrition, and animal welfare pages
- Contact page
- Modular component-based architecture
- Optimized performance with Next.js

Tech Stack

Frontend
- Next.js 16
- React 19
- Tailwind CSS 4

Deployment
- Vercel

Installation

Clone the repository:

```bash
git clone https://github.com/poudwalsamruddhi-alt/KindRed.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```bash
http://localhost:3000
```

Deployment

The project is deployed using Vercel.

Live URL:
https://kind-red.vercel.app/
